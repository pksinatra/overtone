## ChordLink

Domain:

- `https://www.chordlink.net`

Purpose:

- useful music theory tools
- creative music tools
- music visualization tools
- Pro subscription access through Stripe

Related app:

- ChordLog
- Physical path: `https://www.chordlink.net/tools/chordlog/`
- Public app identity: `https://www.chordlog.net`

## Rockeskolen

Domains:

- `https://www.rockeskolen.com`
- `https://www.rockeskolen.no`

Purpose:

- Norwegian learning platform for students, developing musicians, and schools
- VIP membership rather than Pro branding
- Stripe subscriptions
- Feide login for schools and institutions
- Drupal learning portal at `https://portal.rockeskolen.com`
- WordPress front site as entry point
